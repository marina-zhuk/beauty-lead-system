# Setup

## Требования

- Node.js;
- npm.

## Установка

```bash
npm install
```

Если PowerShell блокирует `npm` из-за execution policy, используйте:

```bash
npm.cmd install
```

## Запуск разработки

```bash
npm run dev
```

или:

```bash
npm.cmd run dev
```

После запуска откройте:

```text
http://localhost:3000
```

## Production build

```bash
npm run build
```

или:

```bash
npm.cmd run build
```

## Проверка формы

1. Откройте главную страницу.
2. Нажмите кнопку `Записаться`.
3. Отправьте пустую форму.
4. Убедитесь, что появились ошибки обязательных полей.
5. Заполните имя, телефон и услугу.
6. Отправьте форму.
7. Убедитесь, что появилось сообщение об успешной отправке.
